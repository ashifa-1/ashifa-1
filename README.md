# Hey, I'm Ashifa

I build software because I enjoy turning uncertainty into execution.

Most of the things I've learned started with me having no idea what I was doing. Whether it was backend systems, databases, deployment workflows, or distributed application architecture, my process has always been the same:

Observe → Break it down → Build → Fail → Learn → Repeat

That mindset has shaped how I approach engineering today.

## My Moat

I turn uncertainty into execution.

When I encounter something unfamiliar, I don't wait until I feel completely ready. I start with what I know, learn what I don't, and keep iterating until I can build with confidence.

Many people avoid things they don't understand.

I tend to move toward them.
